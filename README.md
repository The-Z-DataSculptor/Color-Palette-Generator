# 🎨 Color Palette Generator

A responsive web-based Color Palette Generator with a custom paint-brush cursor and "Copy to Clipboard" functionality. Built with HTML, CSS, and JavaScript.

## 🚀 Features
* **Random Generation:** Create 5 unique colors with one click.
* **Copy to Clipboard:** Click the 📋 icon to instantly copy the Hex code.
* **Custom Cursor:** Features a thematic paint-brush cursor for an immersive experience.
* **Interactive UI:** Smooth hover effects and "lifting" card animations.

## 🛠️ Tech Stack
* **HTML5, CSS3, JavaScript (ES6)**
